# Superstore SQL Analysis

## Project Overview
SQL analysis of 9,994 rows of retail sales data using SQLite.
This project explores the same dataset as my Power BI dashboard,
answering business questions using SQL queries.

## Dataset
Sample Superstore — public retail dataset with 9,994 rows covering
orders, sales, profit, regions, and product categories.

## Tools Used
- SQL (SQLite)
- sqliteonline.com

## Business Questions Answered

1. How many total orders are in the dataset?
2. What do the orders from the West region look like?
3. What is the total sales figure from the West region?
4. What are the total sales for every region?
5. What are the total profits for every region?
6. Which region generates the most sales, ranked highest to lowest?
7. Which region generates the most profit, ranked highest to lowest?
8. Which are the top 5 most profitable sub-categories?
9. Does the Sales column have any missing values?

## Key Findings
- West region leads in both sales and profit
- Central has the second highest sales but the lowest profit
- Top 5 most profitable sub-categories: Copiers, Phones, Accessories, Paper, Binders
- Zero missing values confirmed in the Sales column
- Findings consistent with Power BI dashboard analysis of the same dataset

## Related Project
This SQL analysis connects directly to my Power BI Superstore Dashboard
where the same findings were visualised interactively.
